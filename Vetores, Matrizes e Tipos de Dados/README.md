# Vetores, Matrizes e Tipos de Dados

## Tipos de Dados
  
  | Tipo  | Descrição | Exemplo |
  | ------------- | ------------- | :-------------: |
  | `int`           | Inteiro       | 39 |
  | `float`         | Ponto Flutuante, 7 dígitos de precisão  | 3.1415|
  | `double`        | Ponto Flutuante Duplo, 15 dígitos de precisão  | 3.1415926 |
  | `char`          | Caractere       | 'a' |
